# Running the R App

1. **Open R or RStudio.**

2. **Set the working directory:** 
  ```r
  setwd("/Users/jake/Desktop/Projects/branding-package/playground/jake/linked-view-interactions")
  ```

3. **Run the app:**

  ```r
  library(shiny)
  runApp()
  ```

The app should now launch in your default web browser.
